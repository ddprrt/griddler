griddler

Bootstrap grids without the clutter

Why griddler?

Don't get us wrong: Bootstrap is great on many occassions and CSS developers surely can learn a lot! However, most of the time Bootstrap is used rather foolishly, dropping in tons of CSS in a website whilst needing just a few components, resulting sometimes in 98% of unused CSS code!

We found out that a lot of the designers are using Bootstrap just for the sake of the grid system. And we decided to extract that system for an easy-to-use drop-in replacement.

But griddler does not stop here. You can use either the whole CSS grid at once, or customize it to include just the parts you really, really need.

Go ahead and try it out!

Getting Started!

You can either download the CSS file and include griddler in your website. If you are using Sass, it's highly recommended to include our reusable component. Install it with Bower: 

bower install --save griddler

Using the grid

The grid originally defines 4 breakpoints:

xs
small screens up to 768px of screen width
sm
small screens starting at 768px
md
medium screens starting at 992px
lg
large screens starting at 1200px

For each of the breakpoints there is a set of twelve column-classes you can use. For example, an element with the class .col-xs-6 will span 6 of 12 columns starting at zero. If you add .col-sm-3, it will span 3 of 12 columns starting at 768px.

Demo

In the first example we define column spans for all of the screen resolutions. This code 

<div class="container">
  <div class="row">
    <div class="col-xs-12 col-sm-6 col-md-3 col-lg-2">...</div>
    <div class="col-xs-12 col-sm-6 col-md-3 col-lg-2">...</div>
    <div class="col-xs-12 col-sm-6 col-md-3 col-lg-2">...</div>
    <div class="col-xs-12 col-sm-6 col-md-3 col-lg-2">...</div>
    <div class="col-xs-12 col-sm-6 col-md-3 col-lg-2">...</div>
    <div class="col-xs-12 col-sm-6 col-md-3 col-lg-2">...</div>
  </div>
</div>

In the next example we intentionally leave out some breakpoint definitions. This code 

<div class="container">
  <div class="row">
    <div class="col-xs-6 col-lg-3">...</div>
    <div class="col-xs-6 col-lg-3">...</div>
    <div class="col-xs-6 col-lg-3">...</div>
    <div class="col-xs-6 col-lg-3">...</div>

col-xs-6 col-lg-3
col-xs-6 col-lg-3
col-xs-6 col-lg-3

Optimizing your Output

Bootstrap's grid includes a lot of helpers which aren't needed in the most cases. You can push, pull or offset some columns, but most of the time the arrangement is just simple and straightforward. To remove said helper classes, set some flags before including griddler 

$use-offset: false;
$use-puller: false;
$use-pusher: false;

@import "../bower_components/griddler/lib/griddler";

It's also possible to override any of the known variables from Bootsrap. Refer to the docs for more information
